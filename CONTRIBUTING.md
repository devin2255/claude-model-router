# Contributing to claude-model-router

Thank you for your interest in contributing to claude-model-router!

## Development Setup

1. Clone the repository:
```bash
git clone https://github.com/yourusername/claude-model-router.git
cd claude-model-router
```

2. Install in development mode:
```bash
pip install -e .
```

## Code Style

- Follow PEP 8 for Python code
- Add type hints where appropriate
- Write docstrings for functions and modules

## Testing

Run tests before submitting PR:
```bash
python -m pytest tests/
```

## Pull Request Process

1. Fork the repository
2. Create a feature branch (`git checkout -b feature/amazing-feature`)
3. Commit your changes (`git commit -m 'Add amazing feature'`)
4. Push to the branch (`git push origin feature/amazing-feature`)
5. Open a Pull Request

## Reporting Issues

Please include:
- Windows version
- Python version
- Steps to reproduce
- Expected vs actual behavior
