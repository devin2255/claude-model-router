# model-router 项目文档

> Windows CLI 工具，用于在 Kimi 和 OpenAI 模型之间快速切换，并自动管理相关环境变量。

## 项目简介

model-router 是一个专为 Windows 环境设计的命令行工具，主要解决以下问题：

- **模型切换**：在 Kimi（Moonshot）和 OpenAI 模型之间快速切换
- **环境管理**：自动更新 Windows 系统/用户环境变量
- **协议转换**：内置代理服务器，将 Anthropic API 协议转换为 OpenAI 兼容协议
- **智能检测**：根据 IP 地理位置自动选择最佳模型

## 快速开始

```bat
# 切换到 Kimi 模型
model-router kimi

# 切换到 OpenAI 模型
model-router openai

# 自动检测并选择模型
model-router model

# 启动 Claude CLI
model-router claude
```
